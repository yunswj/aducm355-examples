# 循环伏安法上位机软件

这是一个用于ADuCM355循环伏安法项目的上位机软件，提供了完整的数据采集、可视化和分析功能。

## 功能特性

- **串口通信**: 支持与ADuCM355设备的UART通信
- **参数配置**: 可配置电压范围、扫描速率、RTIA等参数
- **实时数据显示**: 实时显示CV曲线、时间序列等图表
- **数据导出**: 支持CSV、JSON、XML、TXT、Excel等多种格式导出
- **用户友好界面**: 基于PyQt5的现代化界面设计

## 系统要求

- Python 3.7+
- Windows/Linux/macOS
- 串口设备

## 使用说明

### 1. 连接设备

1. 启动程序后，在"连接设置"面板中选择正确的串口
2. 设置波特率（默认115200）
3. 点击"连接"按钮建立通信

### 2. 配置参数

在"参数配置"面板中设置：
- 电压范围（起始电压、结束电压、步进电压）
- 扫描参数（扫描速率、循环次数）
- 电流范围和RTIA设置
- 测量模式选择

### 3. 开始测量

1. 在"测量控制"面板中点击"开始测量"
2. 实时观察数据可视化面板中的曲线变化
3. 可以随时暂停或停止测量

### 4. 数据导出

- 在数据可视化面板中点击"导出数据"按钮
- 选择导出格式（CSV、JSON、XML、TXT、Excel）
- 选择保存路径完成导出

## 通信协议

程序支持以下UART命令格式：

### 基本命令格式
```
$<命令>,<参数1>,<参数2>,...*<校验和>
```

### 主要命令

- `$SVR,start,peak*CS` - 设置电压范围
- `$SCA,rtia_index*CS` - 设置RTIA索引
- `$SSR,rate*CS` - 设置扫描速率 (V/s)
- `$SSP,steps,duration*CS` - 设置步数和总持续时间
- `$SMD,mode*CS` - 切换测量模式

### 响应格式
- 成功: `$ACK*CS`
- 错误: `$NAK,<错误码>*CS`
- 数据: `$DATA,<电压>,<电流>*CS`

## 故障排除

### 常见问题

1. **无法连接串口**
   - 检查设备是否正确连接
   - 确认串口号和波特率设置
   - 检查串口是否被其他程序占用

2. **数据显示异常**
   - 检查通信协议是否匹配
   - 确认设备固件版本兼容性

3. **导出失败**
   - 检查文件路径权限
   - 确认磁盘空间充足
